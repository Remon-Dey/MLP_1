# Insurance Cost Prediction using Linear Regression

This project focuses on predicting insurance charges using Machine Learning. The workflow begins with comprehensive data preprocessing and feature engineering to transform raw data into a clean, model-ready format. After preprocessing, a Linear Regression model is trained to predict insurance costs based on customer attributes.

## Project Workflow

* Exploratory Data Analysis (EDA)
* Data Cleaning and Preprocessing
* Missing Value Handling
* Outlier Detection and Treatment
* Label Encoding
* One-Hot Encoding
* Feature Engineering and Extraction
* Data Transformation and Scaling
* Linear Regression Model Training
* Model Evaluation and Performance Analysis

## Objective

The primary objective of this project is to build a reliable insurance cost prediction model by applying proper data preprocessing techniques and training a Linear Regression algorithm. The project demonstrates an end-to-end Machine Learning workflow, from raw data preparation to predictive modeling and evaluation.
