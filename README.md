# Machine Learning Data Preprocessing Pipeline

This project focuses on preparing raw datasets for machine learning by performing all essential preprocessing steps before model training. The workflow includes:

* Exploratory Data Analysis (EDA)
* Data Cleaning and Preprocessing
* Missing Value Treatment
* Outlier Detection and Handling
* Label Encoding
* One-Hot Encoding
* Feature Engineering
* Feature Extraction
* Data Transformation and Scaling

The goal is to transform raw data into a clean, structured, and model-ready format, ensuring better performance and reliability of machine learning algorithms.
